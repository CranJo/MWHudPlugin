# 🔌 MWHudPlugin — Server-Side Data Provider

The server-side companion plugin for the **MWHud Fabric Mod**. Collects real-time player statistics and pushes them to connected clients over a custom network channel.

---

## 📖 About

**MWHud Plugin** is a Paper/Spigot server plugin that gathers player data through PlaceholderAPI and broadcasts it to players running the [MWHud Mod](https://modrinth.com/mod/mwhud).

Once installed, the plugin automatically detects when a player joins with the mod installed and begins sending live updates — no commands, no configuration, just plug and play.

---

## 📦 Installation

1. Download the latest `MWHudPlugin-*.jar` from the releases page
2. Drop it into your server's `plugins/` folder
3. Ensure [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/) is installed
4. Restart or reload your server

---

## 📊 Data Provided

| Stat | MWHudPlugin + PlaceholderAPI Dependency |
|------|:---:|
| Money | Required |
| Mobcoins | Required |
| Fly Time | Required |
| Claim Blocks | Required |
| PvP Status | Required |
| Envoy Timer | Required |
| Key-for-all Timer | Required |
| Party Key | Required |
| Online Players | Required |
| Server Title | Required |

---

## ✅ Requirements

- Paper / Spigot **1.21 – 1.21.11**
- [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/)
- [MWHud Mod](https://modrinth.com/mod/mwhud) (client-side)
